OBD1 Scanner — USB Flasher (recovery)

Use this if over-the-air (Wi-Fi) updates aren't working. It re-flashes the entire device over USB and always recovers it. Requires a desktop Chrome, Edge, or Opera browser (Safari, Firefox, and phones don't support USB flashing from the browser).

Flash firmware over USB

Pick the firmware version, then connect and flash. The full-flash image is downloaded from that version's GitHub release.

Your browser can't flash over USB — use desktop Chrome/Edge. Serial access was blocked. Reload and allow the port.

How to flash

1. Plug the scanner into the computer with a USB cable. 2. Pick a version and click Connect & Flash. 3. Pick the device's serial port (ESP32-S3 / USB JTAG/serial). 4. Confirm — it erases and reflashes, then reboots.

After flashing, the scanner broadcasts the OBD1_Scanner Wi-Fi again and OTA updates should work.
