OBD1 Scanner — USB Flasher (recovery)

Over-the-air update (download .ota) →
Use this if over-the-air (Wi-Fi) updates aren't working. It re-flashes the entire device over USB and always recovers it. Requires a desktop Chrome, Edge, or Opera browser (Safari, Firefox, and phones don't support USB flashing from the browser).

How to flash

1. Plug the scanner into the computer with a USB cable. 2. Click a Connect & Flash button. 3. Pick the device's serial port (ESP32-S3 / USB JTAG/serial). 4. Confirm — it erases and reflashes, then reboots.

After flashing, the scanner broadcasts the OBD1_Scanner Wi-Fi again and OTA updates should work.
